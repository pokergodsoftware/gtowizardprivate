# Áudios do Timebank

Este diretório deve conter os seguintes arquivos de áudio:

## Arquivos Necessários:

1. **timebank1.mp3**
   - Toca quando o timebank chega em 8 segundos
   - Som de alerta suave
   - Formato: MP3
   - Duração recomendada: 0.5-1 segundo

2. **timebank2.mp3**
   - Toca quando o timebank chega em 3 segundos
   - Som de alerta mais urgente
   - Formato: MP3
   - Duração recomendada: 0.5-1 segundo

## Como Adicionar os Áudios:

1. Grave ou baixe os sons de alerta
2. Converta para formato MP3
3. Coloque os arquivos neste diretório com os nomes exatos:
   - `timebank1.mp3`
   - `timebank2.mp3`

## Sugestões de Sons:

- **timebank1.mp3**: Som de "beep" simples ou "tick" de relógio
- **timebank2.mp3**: Som de "beep beep beep" mais rápido ou alarme urgente

## Alternativa Temporária:

Se não tiver os áudios, o sistema funcionará normalmente sem eles. Os áudios são opcionais e o timebank visual continuará funcionando.
